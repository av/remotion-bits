## How to Use Bits

Each bit in this catalog includes:
- **Live Preview**: Interactive player with playback controls to see the animation in action
- **Source Code**: View and optionally edit the implementation code
- **Metadata**: Duration, dimensions, and tags for easy reference

### Installation Methods

**Option 1: Install the full package**
```bash
npm install remotion-bits
```

Then import and use components directly:
```tsx
import { TextTransition, BackgroundTransition } from 'remotion-bits';
```

**Option 2: Use jsrepo to add specific components**
```bash
npx jsrepo add text-transition
npx jsrepo add background-transition
```

### Customization

All bits are fully customizable. You can:
- Adjust animation timing and duration
- Modify colors, gradients, and styles
- Change easing functions and transitions
- Combine multiple bits for complex effects

Copy the code from any playground and adapt it to your specific needs!
