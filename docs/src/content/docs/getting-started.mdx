Remotion Bits is a collection of beautiful, production-ready animation components for [Remotion](https://remotion.dev). Build complex video animations with elegant React components, pre-built effects, and a powerful utility library.

## Prerequisites

Remotion Bits requires the following peer dependencies to already be installed:

- `remotion` >= 4.0.0
- `react` >= 18.0.0
- `react-dom` >= 18.0.0

<Aside>
Don't have Remotion set up yet? Follow the [Remotion quickstart guide](https://www.remotion.dev/docs/) to get started.
</Aside>

## Core Components

Remotion Bits provides five powerful animation components:

### Text Animations

[**AnimatedText**](/docs/reference/animated-text) lets you animate text character-by-character, word-by-word, or by line with configurable staggering and easing.

<HeroShowcase client:only="react" bitNames={["FadeIn", "WordByWord", "CharByChar"]} layout="row" />

### Motion Effects

[**StaggeredMotion**](/docs/reference/staggered-motion) applies coordinated animations across multiple child elements with directional timing control.

<HeroShowcase client:only="react" bitNames={["BlurSlideWord", "SlideFromLeft"]} layout="row" />

### Gradient Transitions

[**GradientTransition**](/docs/reference/gradient-transition) smoothly interpolates between CSS gradients with support for linear, radial, and conic gradients.

<HeroShowcase client:only="react" bitNames={["LinearGradient", "RadialGradient", "ConicGradient"]} layout="row" />

### Particles

[**ParticleSystem**](/docs/reference/particle-system) creates complex particle effects with customizable behaviors like gravity, drag, and wiggle animations.

<HeroShowcase client:only="react" bitNames={["ParticlesSnow", "ParticlesFountain", "ParticlesGrid"]} layout="row" />

### 3D Scenes

[**Scene3D**](/docs/reference/scene3d) renders interactive 3D scenes with camera controls, sequential animations via steps, and 3D element transformations.

<HeroShowcase client:only="react" bitNames={["Scene3DPresentation", "Elements3D", "FlyingThroughWords"]} layout="row" />

## Quick Start

Here's a simple example animating text with fade and slide effects:

```tsx title="src/MyComposition.tsx"
import { Composition } from 'remotion';
import { AnimatedText } from 'remotion-bits';

export const MyComposition = () => {
  return (
    <div style={{
      backgroundColor: '#000',
      width: '100%',
      height: '100%',
      display: 'flex',
      alignItems: 'center',
      justifyContent: 'center'
    }}>
      <AnimatedText
        transition={{
          split: 'word',
          opacity: [0, 1],
          y: [50, 0],
          stagger: 3,
          duration: 30
        }}
        style={{
          fontSize: '4rem',
          color: 'white',
          fontWeight: 'bold'
        }}
      >
        Hello Remotion Bits!
      </AnimatedText>
    </div>
  );
};

// Register the composition
export const Root = () => {
  return (
    <Composition
      id="MyComposition"
      component={MyComposition}
      durationInFrames={120}
      fps={30}
      width={1920}
      height={1080}
    />
  );
};
```

## Using Utilities

Remotion Bits includes powerful utility functions for advanced animations:

```tsx title="Using interpolation utilities"
import { interpolate } from 'remotion-bits';

// Interpolate with easing functions
const scale = interpolate(frame, 0, 60, 0.5, 1.5, {
  easing: 'easeOutBounce'
});
```

<Aside type="tip">
Check out the [Utilities Reference](/docs/reference/utils-interpolation) for color interpolation, geometry handling, motion calculations, and more.
</Aside>

## TypeScript Support

Remotion Bits is built with TypeScript and provides full type definitions out of the box:

```tsx title="Full type safety"
import type { AnimatedText } from 'remotion-bits';
import { AnimatedText as AnimatedTextComponent } from 'remotion-bits';

const MyText: React.FC = () => {
  return (
    <AnimatedTextComponent
      transition={{
        split: 'character',
        opacity: [0, 1],
        rotate: [45, 0],
        duration: 40,
        easing: 'easeOutQuad'
      }}
    >
      Fully typed!
    </AnimatedTextComponent>
  );
};
```
