## Overview

This promo video brings together all the major features of Remotion Bits into a single, cohesive composition. It's designed to showcase what's possible when you combine the library's components, utilities, and hooks.

## Features Demonstrated

### AnimatedText
Sophisticated text animations with granular control over character timing, easing, and transforms.

### GradientTransition
Smooth, visually stunning gradient backgrounds that transition between colors and styles.

### StaggeredMotion
Orchestrate complex animations with precise timing control across multiple elements.

### Particle System
Dynamic particle effects using `Particles`, `Spawner`, and `Behavior` components for physics-based motion.

### Scene3D
Full 3D scene composition with camera controls, element positioning, and transform interpolation.

### useViewportRect
Responsive sizing and layout that adapts to any video dimensions.

## Usage

This composition serves as both a promo video and a reference implementation. Use it to:

- See how different components work together
- Learn best practices for composition structure
- Get inspiration for your own videos
- Test the full capabilities of Remotion Bits

## Technical Notes

The showcase is built to be modular and extensible. Each feature section is carefully timed and transitions smoothly into the next, demonstrating how to build professional-quality video compositions with Remotion Bits.
