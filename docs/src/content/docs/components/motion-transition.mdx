# MotionTransition

The `MotionTransition` component is a flexible wrapper that can animate any React element with smooth transitions. It's the foundation that powers TextTransition and can be used directly for custom animations.

## Installation

```bash
npm install remotion-bits
```

Or via jsrepo:

```bash
npx jsrepo add motion-transition
```

## Basic Usage

```tsx
import { MotionTransition } from 'remotion-bits';

export const MyComponent = () => (
  <MotionTransition
    transition={{
      opacity: [0, 1],
      scale: [0.8, 1],
      rotate: [0, 360]
    }}
  >
    <div>Animated Content</div>
  </MotionTransition>
);
```

## Examples

### Simple Fade & Scale

```tsx
<MotionTransition
  transition={{
    opacity: [0, 1],
    scale: [0.5, 1],
    frames: [0, 30],
    easing: "easeOutQuad"
  }}
>
  <img src="/logo.png" alt="Logo" />
</MotionTransition>
```

### Complex Movement

```tsx
<MotionTransition
  transition={{
    x: [-100, 100, 0],
    y: [0, -50, 0],
    rotate: [0, 180, 360],
    scale: [1, 1.5, 1],
    opacity: [0, 1, 1],
    frames: [0, 45, 90],
    easing: "easeInOutCubic"
  }}
>
  <div className="animated-box">
    Content
  </div>
</MotionTransition>
```

### Staggered Children

```tsx
const items = ['Item 1', 'Item 2', 'Item 3'];

{items.map((item, i) => (
  <MotionTransition
    key={i}
    transition={{
      opacity: [0, 1],
      y: [50, 0],
      frames: [i * 10, i * 10 + 30],
      easing: "easeOutQuad"
    }}
  >
    <div>{item}</div>
  </MotionTransition>
))}
```

### Image Entrance

```tsx
<MotionTransition
  transition={{
    opacity: [0, 1],
    scale: [1.2, 1],
    blur: [10, 0], // Blur effect
    frames: [30, 60],
    easing: "easeOut"
  }}
>
  <img
    src="/hero-image.jpg"
    alt="Hero"
    style={{ width: '100%', height: 'auto' }}
  />
</MotionTransition>
```

## Props

### `transition`

Configuration object for the animation.

#### Transform Properties

| Property | Type | Description |
|----------|------|-------------|
| `x` | `number[]` | Horizontal translation in pixels |
| `y` | `number[]` | Vertical translation in pixels |
| `scale` | `number[]` | Scale transformation (1 = 100%) |
| `scaleX` | `number[]` | Horizontal scale only |
| `scaleY` | `number[]` | Vertical scale only |
| `rotate` | `number[]` | Rotation in degrees |
| `rotateX` | `number[]` | 3D rotation on X axis |
| `rotateY` | `number[]` | 3D rotation on Y axis |
| `rotateZ` | `number[]` | 3D rotation on Z axis |
| `skewX` | `number[]` | Horizontal skew in degrees |
| `skewY` | `number[]` | Vertical skew in degrees |

#### Visual Properties

| Property | Type | Description |
|----------|------|-------------|
| `opacity` | `number[]` | Opacity (0-1) |
| `blur` | `number[]` | Blur amount in pixels |
| `brightness` | `number[]` | Brightness (1 = 100%) |
| `contrast` | `number[]` | Contrast (1 = 100%) |
| `saturate` | `number[]` | Saturation (1 = 100%) |
| `hueRotate` | `number[]` | Hue rotation in degrees |

#### Timing

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `frames` | `[number, number]` or `number[]` | `[0, 30]` | Animation frame range. Can be `[start, end]` or keyframes `[0, 30, 60]` |
| `easing` | `string \| function` | `"linear"` | Easing function |

### `children`

The React element(s) to animate. Must be a single element.

### `style`

Additional CSS styles to apply to the wrapper.

### `className`

CSS class name for the wrapper element.

## Advanced Usage

### 3D Transforms

Create 3D rotation effects:

```tsx
<MotionTransition
  transition={{
    rotateY: [0, 360],
    scale: [1, 1.2, 1],
    frames: [0, 90]
  }}
  style={{ perspective: '1000px' }}
>
  <div className="card">
    3D Rotating Card
  </div>
</MotionTransition>
```

### Filter Effects

Combine multiple filter effects:

```tsx
<MotionTransition
  transition={{
    blur: [10, 0],
    brightness: [0.5, 1],
    saturate: [0, 1],
    frames: [0, 60],
    easing: "easeOut"
  }}
>
  <img src="/photo.jpg" alt="Photo" />
</MotionTransition>
```

### Multi-Stage Animation

Use multiple keyframes for complex animations:

```tsx
<MotionTransition
  transition={{
    x: [0, 200, 200, 0],
    y: [0, 0, 200, 200],
    rotate: [0, 90, 180, 270],
    frames: [0, 30, 60, 90, 120],
    easing: "easeInOutQuad"
  }}
>
  <div className="box">Moving Box</div>
</MotionTransition>
```

### Custom Easing Per Property

```tsx
<MotionTransition
  transition={{
    opacity: {
      values: [0, 1],
      easing: "easeIn"
    },
    y: {
      values: [100, 0],
      easing: "easeOutBounce"
    },
    frames: [0, 60]
  }}
>
  <div>Custom Easing</div>
</MotionTransition>
```

## Performance Tips

1. **Transform over position**: Use `x`/`y` transforms instead of `left`/`top` for better performance
2. **Opacity**: Animating opacity is very performant
3. **GPU acceleration**: Transform and opacity animations are GPU-accelerated
4. **Avoid layout shifts**: Use transforms that don't affect document flow

## Comparison with Other Components

- **TextTransition**: Built on MotionTransition but adds text-specific features (split, stagger, color)
- **BackgroundTransition**: Specialized for gradient backgrounds
- **MotionTransition**: Most flexible, works with any content

Use MotionTransition when you need:
- Full control over animation properties
- To animate non-text elements
- Custom animation sequences
- Filter effects
