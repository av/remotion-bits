# TextTransition

The `TextTransition` component provides powerful text animation capabilities with support for word-by-word, character-by-character, and line-by-line transitions.

## Installation

```bash
npm install remotion-bits
```

Or via jsrepo:

```bash
npx jsrepo add text-transition
```

## Basic Usage

```tsx
import { TextTransition } from 'remotion-bits';

export const MyComponent = () => (
  <TextTransition
    transition={{
      opacity: [0, 1],
      y: [50, 0]
    }}
  >
    Hello World
  </TextTransition>
);
```

## Props

### `transition`

Configuration object for the text animation.

#### Animatable Properties

| Property | Type | Description |
|----------|------|-------------|
| `opacity` | `number[]` | Array of opacity values (0-1) |
| `x` | `number[]` | Horizontal translation in pixels |
| `y` | `number[]` | Vertical translation in pixels |
| `scale` | `number[]` | Scale transformation |
| `rotate` | `number[]` | Rotation in degrees |
| `color` | `string[]` | Array of color values (any CSS color format) |

#### Timing & Easing

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `frames` | `[number, number]` | `[0, 30]` | Start and end frame for the animation |
| `easing` | `string \| function` | `"linear"` | Easing function (see [Easing](/docs/utilities/easing)) |

#### Text Splitting

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `split` | `"word" \| "character" \| "line"` | - | How to split the text for staggered animations |
| `splitStagger` | `number` | `0` | Frame delay between each split unit |

#### Cycling

| Property | Type | Description |
|----------|------|-------------|
| `cycle.texts` | `string[]` | Array of texts to cycle through |
| `cycle.itemDuration` | `number` | Duration in frames for each text item |

### `children`

The text content to animate. Can be a string or React nodes.

### `style`

Standard React CSS properties to apply to the text element.

### `className`

CSS class name for the wrapper element.

## Advanced Usage

### Custom Easing

Use a custom easing function for precise control:

```tsx
<TextTransition
  transition={{
    opacity: [0, 1],
    easing: (t) => t * t * t // cubic ease
  }}
>
  Custom Easing
</TextTransition>
```

### Keyframe Animation

Use multiple values for complex keyframe animations:

```tsx
<TextTransition
  transition={{
    y: [100, -20, 0],
    opacity: [0, 1, 1],
    frames: [0, 40, 60]
  }}
>
  Bouncing Text
</TextTransition>
```

### Color Interpolation

Supports multiple color formats including hex, rgb, hsl, and oklch:

```tsx
<TextTransition
  transition={{
    color: [
      "#ff0000",
      "rgb(0, 255, 0)",
      "hsl(240, 100%, 50%)",
      "oklch(0.7 0.2 270)"
    ]
  }}
>
  Rainbow Text
</TextTransition>
```

## Tips

1. **Performance**: Character-level splitting creates more DOM elements. Use word or line splitting for longer text.

2. **Stagger timing**: Adjust `splitStagger` based on your FPS. At 30fps, a value of 3 = 0.1 seconds.

3. **Frame range**: Use the `frames` property to control when the animation starts and ends within your composition.

4. **Easing functions**: See the [Easing guide](/docs/utilities/easing) for available presets.
