# Background Effect Examples

Create stunning visual backgrounds with smooth gradient transitions using Remotion Bits.

## Basic Gradients

### Linear Gradient Transition

Classic linear gradient transition between two color schemes.

<ShowcasePlayer client:only="react" component={LinearGradientShowcase} duration={90} />

```tsx
<BackgroundTransition
  gradient={[
    "linear-gradient(0deg, #667eea 0%, #764ba2 100%)",
    "linear-gradient(180deg, #f093fb 0%, #f5576c 100%)"
  ]}
  duration={90}
>
  <YourContent />
</BackgroundTransition>
```

**Use cases:**
- Section transitions
- Mood changes
- Time progression (day to night)

## Practical Examples

### Video Chapter Transitions

```tsx
export const ChapterTransition = ({ fromChapter, toChapter }: Props) => {
  const chapterGradients = {
    intro: "linear-gradient(135deg, #667eea 0%, #764ba2 100%)",
    content: "linear-gradient(135deg, #f093fb 0%, #f5576c 100%)",
    conclusion: "linear-gradient(135deg, #4facfe 0%, #00f2fe 100%)"
  };

  return (
    <BackgroundTransition
      gradient={[
        chapterGradients[fromChapter],
        chapterGradients[toChapter]
      ]}
      duration={60}
      easing="easeInOutCubic"
    >
      <div style={{
        display: 'flex',
        alignItems: 'center',
        justifyContent: 'center',
        fontSize: '4rem',
        color: 'white',
        fontWeight: 'bold'
      }}>
        Chapter {toChapter}
      </div>
    </BackgroundTransition>
  );
};
```

### Time of Day Background

```tsx

  const frame = useCurrentFrame();

  // Cycle through day times
  const gradients = [
    "linear-gradient(180deg, #000428 0%, #004e92 100%)", // Night

    "linear-gradient(180deg, #56CCF2 0%, #2F80ED 100%)", // Day
    "linear-gradient(180deg, #F2994A 0%, #F2C94C 100%)", // Sunset
    "linear-gradient(180deg, #000428 0%, #004e92 100%)"  // Night

  return (
    <BackgroundTransition

        display: 'flex',

        justifyContent: 'center',

        fontSize: '3rem'

        24 Hour Time Lapse
      </div>

```

## Performance Tips

1. **GPU Acceleration**: Gradient animations are GPU-accelerated for smooth performance

2. **Optimize Stops**: Fewer color stops = better performance

3. **Reuse Gradients**: Define gradients as constants to avoid recreating them

4. **Consider Duration**: Longer transitions need fewer FPS for smooth appearance
