# Text Animation Examples

Explore practical examples and patterns for creating beautiful text animations with Remotion Bits.

## Title Sequences

### Hero Title with Stagger

Perfect for video intros and hero sections.


## Practical Patterns

### Video Intro Sequence

```tsx
export const VideoIntro = () => {
  return (
    <AbsoluteFill style={{ backgroundColor: '#000' }}>
      {/* Title appears first */}
      <TextTransition
        transition={{
          opacity: [0, 1, 1, 0],
          scale: [0.8, 1, 1, 0.8],
          frames: [0, 20, 80, 100]
        }}
        style={{
          fontSize: '6rem',
          color: 'white',
          position: 'absolute',
          top: '40%',
          width: '100%',
          textAlign: 'center'
        }}
      >
        Your Brand Name
      </TextTransition>

      {/* Subtitle follows */}
      <TextTransition
        transition={{
          opacity: [0, 1, 1, 0],
          y: [30, 0, 0, -30],
          frames: [30, 50, 100, 120]
        }}
        style={{
          fontSize: '2rem',
          color: '#888',
          position: 'absolute',
          top: '55%',
          width: '100%',
          textAlign: 'center'
        }}
      >
        Catchy Tagline
      </TextTransition>
    </AbsoluteFill>
  );
};
```

## Tips & Best Practices

### Timing

- **Fast intro (15-20 frames)**: Modern, snappy feel
- **Medium (30-40 frames)**: Balanced, professional
- **Slow (50+ frames)**: Elegant, dramatic

### Stagger Amount

At 30 FPS:
- 1-2 frames: Very fast, suited for short words
- 3-5 frames: Standard, readable pace
- 10+ frames: Dramatic emphasis on each element

### Font Choices

- **Bold weights**: Work best with animations
- **Sans-serif**: Generally more readable in motion
- **Avoid thin fonts**: Can be hard to read during transitions

### Performance

- Keep `split: "character"` for shorter text (< 20 characters)
- Use `split: "word"` for longer text
- Avoid excessive simultaneous text animations

### Accessibility

- Ensure sufficient contrast ratios
- Don't rely solely on color changes
- Provide alternative static text for screen readers
